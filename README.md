# FUTURE_DS_01 - Business Sales Analysis Dashboard

## Project Overview
This project analyzes business sales data using Power BI to identify revenue trends, profitable categories, top-selling products, and high-performing regions.

## Objective
The objective of this project is to help businesses make data-driven decisions by analyzing sales performance and identifying opportunities for growth.

## Tools Used
- Power BI
- Microsoft Excel
- Superstore Dataset

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Quantity Sold KPI
- Sales Trend Over Time
- Sales by Category
- Profit by Region
- Top 10 Products by Sales
- Region Filter
- Category Filter

## Key Insights
1. Total Sales reached 2.30M.
2. Total Profit reached 286.40K.
3. Technology generated the highest sales among all categories.
4. The West region produced the highest profit.
5. A small number of products contributed significantly to overall revenue.

## Business Recommendations
1. Increase focus on Technology products.
2. Expand business operations in the West region.
3. Maintain inventory for top-selling products.
4. Improve performance of lower-profit categories.
5. Use sales trends for planning promotions and inventory management.
